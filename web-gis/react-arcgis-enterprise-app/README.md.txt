# React ArcGIS Enterprise App

## Overview
A React-based web GIS application that consumes ArcGIS REST Services published from ArcGIS Enterprise.

## What this repository covers
- Web map and service integration
- ArcGIS REST consumption in a React application
- Layer filtering and querying
- Service-driven UI patterns
- Basic GIS application architecture

## Tools and Technologies
- React
- ArcGIS REST Services
- ArcGIS Enterprise
- ArcGIS Pro
- JavaScript

## Repository Structure
- `client/` - application source code
- `architecture/` - diagrams and solution notes
- `docs/` - setup and implementation notes
- `screenshots/` - UI and workflow images
- `sample-config/` - sample configuration templates

## Why this repository
This repository showcases how enterprise GIS services can be consumed in a modern web application.
