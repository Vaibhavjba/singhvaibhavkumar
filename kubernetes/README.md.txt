# Kubernetes Python API Ops

## Overview
A collection of practical Kubernetes Python API scripts for operational visibility and automation.

## What this repository covers
- Pod and workload health checks
- Namespace resource summaries
- Deployment status reporting
- ConfigMap and Secret inventory
- Persistent volume claim reporting

## Tools and Technologies
- Kubernetes
- Python
- Kubernetes Python Client

## Repository Structure
- `cluster-health/` - health-related scripts and summaries
- `namespace-reports/` - namespace-level reporting tools
- `workload-checks/` - deployment and workload checks
- `configmaps-and-secrets/` - configuration visibility helpers
- `storage-checks/` - storage and PVC-related checks

## Why this repository
This repository highlights practical Kubernetes automation relevant to platform operations and cloud-native environments.
